# Changelog

All notable changes to `tenantwave/broadcast` will be documented in this file.

## [Unreleased]

### Added
- Initial release
- Automatic channel scoping for multi-tenant applications
- Strict authorization enforcement
- Background broadcasting support
- TenantBroadcast facade for easy usage
- Configurable tenant key and channel prefix
- Support for both PrivateChannel and PresenceChannel
- Tenant context impersonation for background jobs

## [1.0.0] - 2026-02-02

### Added
- First stable release
- Multi-tenant broadcast channel scoping
- Laravel 9.x, 10.x, 11.x, and 12.x support
- PHP 8.1+ support
